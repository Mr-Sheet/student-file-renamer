# 学生作业文件批量命名规范化工具

根据 Excel 学生名单，自动识别文件名中的姓名（支持中文、拼音全拼、首字母简写），按自定义模板批量生成规范命名建议，并提供重命名执行、提交检查、撤销回滚等功能。

## 快速开始

### 方式一：直接运行（推荐）

1. 将 `学生作业命名工具` 文件夹复制到任意位置
2. 双击 `学生作业命名工具.exe`

### 方式二：源码运行

```bash
pip install PySide6 rapidfuzz pypinyin tabulate openpyxl xlrd
python gui.py
```

CLI 模式：

```bash
python main.py              # 预览
python main.py --apply      # 执行重命名
python main.py --undo logs/xxx.csv  # 撤销
```

## 操作流程

```
① 选择文件夹 → ② 选择名单 → ③ 设置模板 → ④ 预览 → ⑤ 执行重命名
```

### 1. 选择文件夹

包含学生作业的文件夹，工具只扫描直接子文件（不递归子文件夹）。

### 2. 选择名单

Excel 文件（`.xlsx` 或 `.xls`），需包含姓名和学号信息。如果表头不是精确的"姓名"/"学号"（如"学生名字"/"学籍编号"），工具会自动弹出列映射向导让你手动指定。

### 3. 设置命名模板

模板使用 `{占位符}` 语法，例如 `{学号}{姓名}{匹配项}`。

| 占位符 | 含义 | 示例 |
|--------|------|------|
| `{学号}` | Excel 中的学号 | `2023001` |
| `{姓名}` | Excel 中的姓名 | `张三` |
| `{原文件名}` | 原文件名（不含扩展名） | `张三实验1` |
| `{扩展名}` | 原扩展名（含点号） | `.docx` |
| `{实验号}` | 自动提取的实验编号 | `实验1` |
| `{匹配项}` | 根据关键词从文件名提取 | 见下方说明 |

**`{匹配项}` 用法**：在"匹配项"输入框填关键词（如"实验3"），工具会在文件名中搜索该关键词及其中英文变体（实验3 / part3 / lab3 / 作业3 / 实验三等），匹配到的文件统一输出为关键词格式。

### 4. 预览匹配结果

点击预览后显示两个视图（点击切换标签）：

- **匹配结果**：每个文件的匹配状态、建议新文件名
  - ✅ 已匹配 —— 识别成功
  - ❌ 未匹配 —— 无法匹配到任何学生
  - ⚠️ 多重匹配 —— 多个学生分数接近，无法自动判定
- **异常情况**：只显示缺交和多交的学生

### 5. 执行重命名

预览确认无误后，点击"执行重命名"，工具会逐文件改名。每步有安全检查（不覆盖已存在文件、捕获异常等），同时生成操作日志。

## 匹配方式

工具按以下优先级自动识别文件属于哪位学生：

| 优先级 | 匹配方式 | 分数 | 示例 |
|--------|---------|------|------|
| 1 | 中文姓名子串匹配 | 100 | `张三实验1.docx` → 张三 |
| 2 | 拼音全拼匹配 | 90 | `zhangsan_homework.pdf` → 张三 |
| 3 | 拼音首字母匹配 | 80 | `zs_test.docx` → 张三 |
| 4 | rapidfuzz 模糊匹配 | 可变 | `zhagnsan.pdf`（少打一个字母）→ 张三 |

可调整匹配阈值（0-100）控制匹配松紧度，默认 80。可关闭拼音匹配（仅中文姓名）。

## 右键菜单

### 匹配结果表

- 编辑建议新文件名 / 复制原文件名 / 复制建议新文件名
- 打开文件所在文件夹
- 选择 / 更改匹配学生
- 手动标记为已匹配 / 未匹配
- 多选时支持批量标记

### 异常情况表

- 复制缺交 / 多交名单（学号 + 姓名）
- 导出异常名单 CSV

## 安全保障

- 默认仅预览，需主动点击"执行重命名"才真正改名
- 目标文件已存在 → 自动跳过
- 原文件不存在 → 自动跳过
- 每次重命名自动生成操作日志（`logs/` 目录下）
- 支持一键撤销上次重命名
- 所有异常均捕获，不会中断程序

## 项目结构

```
├── gui.py              # GUI 入口（PySide6）
├── main.py             # CLI 入口
├── config.json          # 配置文件（GUI 和 CLI 共享）
├── icon.ico             # 程序图标
├── README.md
├── core/
│   ├── matcher.py       # 匹配引擎
│   ├── renamer.py       # 重命名 + 撤销
│   ├── reporter.py      # 终端输出 + CSV 导出
│   └── checker.py       # 提交检查（缺交/多交）
├── students.xlsx        # 学生名单（示例）
├── files/               # 作业文件（示例）
└── logs/                # 操作日志
```

## 打包

```bash
pip install pyinstaller

```
